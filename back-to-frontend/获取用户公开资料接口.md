# 获取用户公开资料接口

> 供前端开发对接使用：图片展示页点击创建者头像/昵称时，跳转创建者个人资料页。

---

## 基本信息

| 项目 | 值 |
|------|-----|
| **请求地址** | `GET /api/user/{id}` |
| **认证方式** | **无需登录**（公共接口） |

---

## 路径参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | `number` | 是 | 用户 ID（即 `PictureVO.userId`） |

---

## 请求示例

```
GET /api/user/1
```

---

## 成功响应 `code: 0`

```json
{
  "code": 0,
  "data": {
    "id": 1,
    "userAccount": "alice01",
    "userName": "爱丽丝",
    "userAvatar": "https://picture-1397275457.cos.ap-nanjing.myqcloud.com/avatar/2026/07/11/abc123.jpg",
    "userProfile": "热爱摄影的工程师",
    "userRole": "user",
    "createTime": "2026-07-10T12:00:00",
    "updateTime": "2026-07-11T10:30:00"
  },
  "message": "ok"
}
```

---

## 失败响应

### 用户不存在

```json
{
  "code": 41002,
  "data": null,
  "message": "用户不存在"
}
```

---

## 响应字段说明

`data` 为 [UserVO](#uservo) 对象。

---

## 数据模型

### UserVO

| 字段 | 类型 | 示例 | 说明 |
|------|------|------|------|
| `id` | `number` | `1` | 用户 ID |
| `userAccount` | `string` | `"alice01"` | 登录账号 |
| `userName` | `string \| null` | `"爱丽丝"` | 昵称 |
| `userAvatar` | `string \| null` | `"https://..."` | 头像 URL |
| `userProfile` | `string \| null` | `"热爱摄影的工程师"` | 个人简介 |
| `userRole` | `string` | `"user"` | 角色：`"user"` / `"admin"` |
| `createTime` | `string` | `"2026-07-10T12:00:00"` | ISO 格式，注册时间 |
| `updateTime` | `string` | `"2026-07-11T10:30:00"` | ISO 格式，最近更新时间 |

---

## 错误码

| code | 说明 |
|------|------|
| 0 | 成功 |
| 41002 | 用户不存在（该用户 ID 可能已注销） |

---

## 前端集成指引

### 使用场景

在图片详情页/图片列表中，每张图片的 `PictureVO` 包含创建者信息：

```json
{
  "id": 1,
  "url": "https://...",
  "name": "my_photo.jpg",
  "userId": 1,
  "user": {
    "id": 1,
    "userName": "爱丽丝",
    "userAvatar": "https://..."
  }
}
```

**交互流程**：

1. 用户点击图片下方的创建者头像或昵称
2. 前端获取 `PictureVO.userId` 或 `PictureVO.user.id`
3. 调用 `GET /api/user/{userId}` 获取完整资料
4. 跳转至创建者个人资料页

### 数据结构说明

- `PictureVO.user`（`UserVO`）已在图库列表返回中携带，**基本展示无需额外请求**
- `GET /api/user/{id}` 用于 **个人资料页** 获取完整信息（如个人简介 `userProfile`），确保数据最新

### TypeScript 类型定义参考

```typescript
interface BaseResponse<T> {
  code: number
  data: T
  message: string
}

interface UserVO {
  id: number
  userAccount: string
  userName: string | null
  userAvatar: string | null
  userProfile: string | null
  userRole: string
  createTime: string
  updateTime: string
}

// 图片列表中的创建者信息（PictureVO.user）
// 结构与 UserVO 一致，已在分页接口中预加载
```

### 代码示例

```typescript
// 跳转创建者个人资料页
async function goToCreatorProfile(creatorId: number) {
  const res = await fetch(`/api/user/${creatorId}`)
  const result: BaseResponse<UserVO> = await res.json()
  if (result.code === 0) {
    // 跳转个人资料页，数据可用 result.data
    navigateTo(`/user/${result.data.id}`, { state: { user: result.data } })
  } else if (result.code === 41002) {
    // 用户不存在（已注销）
    showToast('该用户已注销')
  }
}
```
